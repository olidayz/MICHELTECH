# Video Game Aesthetics

---

## Unreal Engine Cinematic
- **Look:** Photorealistic rendering, ray-traced lighting, volumetric fog, AAA quality, cutscene feel
- **Mood:** Epic, immersive, next-gen, blockbuster
- **Use when:** High-fidelity fantasy/sci-fi, realistic game worlds, virtual production look
- **Prompt language:** "Unreal Engine 5, photorealistic, ray-traced, volumetric lighting, AAA game cinematic, hyperrealistic"

## Retro Pixel (NES/SNES/Arcade)
- **Look:** 8-bit or 16-bit pixel grid, limited palette, sprite-based characters, tile-based backgrounds
- **Mood:** Nostalgia, charm, constraint-as-style, retro gaming
- **Use when:** Retro gaming content, nostalgia marketing, indie game aesthetic, chiptune culture
- **Prompt language:** "NES 8-bit pixel art, retro game, limited color palette, sprite, tile-based, classic arcade"

## Isometric
- **Look:** Fixed camera angle (~30°), no perspective distortion, grid-based, tidy, readable
- **Mood:** Strategic, orderly, playful, architectural, miniature world
- **Use when:** Infographic-meets-game, city builders, architectural visualization, clean design content
- **Prompt language:** "isometric view, isometric game art, fixed angle, grid-based, diorama, miniature world, clean"

## Low-Poly
- **Look:** Visible polygonal faces, flat or simple shading, geometric, intentionally rough 3D
- **Mood:** Minimalist, indie, stylized, approachable, modern retro
- **Use when:** Indie game aesthetic, modern-retro, minimalist design, abstract environments
- **Prompt language:** "low-poly 3D, flat shading, geometric, minimalist, indie game style, visible polygon faces"

## Cel-Shaded
- **Look:** 3D geometry with flat cartoon-like shading, bold outlines, looks 2D but moves 3D
- **Mood:** Playful, illustrative, timeless (doesn't age like photorealistic), comic book
- **Use when:** Animated look with 3D depth, comic/manga adaptation, all-ages content
- **Prompt language:** "cel-shaded, toon shading, bold outlines, flat color, 3D with 2D look, Wind Waker style"
- **Reference:** Zelda: Wind Waker, Borderlands, Jet Set Radio, Spider-Verse

## Vaporwave / Liminal Space
- **Look:** Pastel gradients, Roman busts, 90s tech, empty malls, pool rooms, sunset grids, glitch
- **Mood:** Nostalgia for a future that never happened, eerie calm, internet melancholy, surreal
- **Use when:** Internet culture commentary, surreal shorts, dreamlike content, anti-capitalist aesthetics
- **Prompt language:** "vaporwave aesthetic, pastel gradient, roman bust, 90s computer graphics, sunset grid, empty mall, liminal space"

## Glitch Art
- **Look:** Data corruption, pixel sorting, color channel splitting, digital decay, broken signal
- **Mood:** Error-as-beauty, digital anxiety, breakdown, system failure, cyberpunk adjacent
- **Use when:** Tech horror, digital identity, system breakdown, music videos, transition effects
- **Prompt language:** "glitch art, data corruption, pixel sorting, RGB split, digital artifact, broken signal, databending"

## UI/HUD Overlay
- **Look:** Health bars, mini-maps, quest markers, inventory screens overlaid on reality
- **Mood:** Gamified reality, AR/future tech, commentary on gamification
- **Use when:** Sci-fi with game elements, commentary on gamified culture, AR concepts
- **Prompt language:** "game HUD overlay, health bar, mini-map, quest marker, augmented reality UI, game interface on real world"

## Flat Design / Vector Game
- **Look:** Clean shapes, no shadows/gradients, bold solid colors, geometric
- **Mood:** Modern, clean, mobile-game feel, accessible
- **Use when:** Mobile game aesthetic, clean modern design, UI animation, explainer content
- **Prompt language:** "flat design, vector art, bold solid colors, clean geometric shapes, no shadow, mobile game style"

## PS1 / Early 3D
- **Look:** Low-resolution textures, affine texture warping, vertex jitter, fog, jagged polygons
- **Mood:** Uncanny, eerie, nostalgic unease, liminal, horror through limitation
- **Use when:** Horror (PS1 horror is a whole genre), nostalgia, eerie surrealism, VHS-of-3D feeling
- **Prompt language:** "PS1 style 3D, low poly, affine texture warping, vertex jitter, low resolution, early 3D graphics, 1996"
- **Reference:** Silent Hill, Resident Evil 1, Metal Gear Solid, Puppet Combo games
