# Reference Libraries

Where to find visual references for your projects. Search these when exploring styles or looking for specific shot/composition/look inspiration.

---

## Film & Cinematography References

### Flim.ai
- **What:** AI-powered search for film/TV stills
- **Strength:** Search by mood, color, composition, shot type, cinematic language
- **Best for:** Finding specific looks — "warm, intimate, two-shot, golden hour"
- **URL:** flim.ai
- **Access:** Free tier + Paid

### ShotDeck
- **What:** Largest HD film/commercial stills library, curated by DP Lawrence Sher (Joker)
- **Strength:** Tagged by lens, lighting, equipment, cinematographer, film
- **Best for:** Technical reference — "wide shot, anamorphic, golden hour, 50mm lens"
- **URL:** shotdeck.com
- **Access:** Paid (professional standard)

### Shot.cafe
- **What:** Free searchable film stills database
- **Strength:** RGB Parade for each shot (study color grading)
- **Best for:** Color study, free alternative to ShotDeck
- **URL:** shot.cafe
- **Access:** Free

### FrameSet
- **What:** Film/TV reference with organization tools
- **Strength:** Good for building moodboards, playlist creation
- **Best for:** Organizing references, moodboard creation
- **URL:** frameset.app
- **Access:** Free + Paid

### Film Grab
- **What:** Free film stills archive
- **Strength:** Massive collection, no paywall
- **Best for:** Quick searching, free reference hunting
- **URL:** film-grab.com
- **Access:** Free

## Reference Workflow

When exploring a style or look:

1. **Start specific:** Flim.ai or ShotDeck if you know what you're looking for
2. **Broad exploration:** Pinterest, Are.na, or FrameSet for open-ended discovery
3. **Cross-reference:** If you find something, search for more like it
4. **Save/organize:** Create boards/collections as you go
5. **Pull examples:** Save 3-5 strongest references to share or reference later

For Style Mixer exploration:
1. Pick two styles
2. Search Flim.ai for each individually
3. Search for combinations
4. Pull examples from any source that matches the vision
